# Memory Sources

This folder preserves external source files used for Calcwiz planning, research, or roadmap work.

## Purpose

`.memory/sources/` is for source preservation, not interpretation.

Use it for:

- external roadmap files
- exported ChatGPT discussions
- research notes received from outside the repo
- source documents that should remain available after local Downloads or chat history changes

## Preservation Rule

Source snapshots should be stored verbatim.

- Do not add metadata headers to snapshot files.
- Do not edit wording, formatting, spelling, or structure inside a snapshot.
- Put capture metadata in `INDEX.md`, not inside the snapshot.
- Use stable placeholders such as `<local-source>` in metadata instead of exact local paths.

## Interpretation Rule

Interpretations, decisions, summaries, and Calcwiz-specific adaptations belong outside this folder, usually in `.memory/research/`, `.memory/decisions.md`, or `.memory/current-state.md`.

If a source inspires product work, the implementation plan must reference the source snapshot and the separate interpretation note.

## Backfill Policy

Historical source backfill is optional and should happen only when a source is still important for future work. New important sources should be captured here before they are interpreted.
