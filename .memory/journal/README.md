# Journal

- `journal/` is the chronological stream of short dated notes.
- Keep entries concise and factual.
- Use the journal for "what changed today" summaries.
- Use `sessions/` when a task needs richer completion or verification detail.
- Historical files may use a file-level `## Historical Attribution` block.
- New entries should start with a compact agent prefix such as `[agent: codex | model: gpt-5.4]`.
