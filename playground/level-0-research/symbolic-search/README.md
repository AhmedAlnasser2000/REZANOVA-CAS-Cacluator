# Symbolic Search Lab

This folder holds executable `PGL3` research for `sym-search-planner-ordering`.

- The lab compares bounded guarded-solver stage orderings over already-shipped solver stages.
- It must not change stable app behavior.
- Corpus definitions, planner orderings, and comparison rules live beside the lab test so reruns stay reproducible.
- Promotion decisions belong in the record at `playground/records/sym-search-planner-ordering.md`, not in generated output files.
